# ELFDecoder
Executable and Linkable Format decoder

## How to build:
```
$ cmake . && make
```

## Project structure:
```
ELFDecoder
├── bin
│   └── simple.bin     - simple elf file
├── doc
│   └── ELF_Format.pdf - ELF format description
├── README.md          - project description
└── main.cpp           - main project source file
```
